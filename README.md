# Password-Checker-and-Expense-Tracker
Passwor Strength Checker
A console program that evaluates password strength by scanning characters and applying rules. Reports a strength score with labels (Weak / Medium / Strong / Very Strong) and gives actionable tips for improvement.
How It Works
The program runs in a loop, prompting the user to enter a password. It evaluates the password against multiple rules, assigns points, and returns a total score with feedback.

For Loop Expense Tracker
A console program that repeatedly prompts the user to add expenses (amount, category, note, optional date) and stores each in a list. The loop runs until the user types 'done' as the category. Later, for loops summarize and print totals by category.
How It Works
The program uses a while loop to collect expenses indefinitely. Each expense is stored as a dictionary with four fields. When the user exits, for loops iterate through the list to generate summaries.

you can run it using any version of Python
